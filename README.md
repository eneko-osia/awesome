# Awesome WM 
My personal set up for awesome wm 4.3

## Dependencies
awesome-freedesktop-git  
imagemagick  
xcompmgr  
xscreensaver  
xterm  

## Optional
discord  
firefox  
git  
openssh  
steam  

## Setup
1. git clone repo into `~/.config/awesome`
2. init and update submodules
3. sudo cp `~/.config/awesome/scripts/*` `/usr/local/bin`
4. login using awesome wm
